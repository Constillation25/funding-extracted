How Constellation25 Works

1. Enter your prompt or complete the 25-question wizard
2. Mercury-1 analyzes requirements and extracts goals
3. Earth-3 generates infrastructure config (Termux/Vercel/Docker)
4. Jupiter-5 builds business logic and revenue model
5. Saturn-6 creates deployment scripts and CI/CD config
6. Download your bundle or one-click deploy
🎯 Pro Tip: Use specific prompts like "Build a React SaaS with Stripe payments and user auth" for best results.

Frequently Asked Questions

Do I need coding experience? ▸
No! The 25-question wizard guides you through requirements. Advanced users can use the prompt box for direct control.
Can I deploy to my own server? ▸
Yes. All deployment bundles include Docker, Termux, and generic server configs. You own 100% of the generated code.
How are my prompts used? ▸
Prompts are processed locally or via your configured Ollama instance. We never train on your data. See our Privacy Policy.
What if I need changes after generation? ▸
All artifacts are editable source code. Use the Preview tool to review, then modify locally before deploying.
Is there an API for programmatic access? ▸
Yes! Pro and Enterprise plans include API access. Contact us for documentation.
